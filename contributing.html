contributing to the open-source-library project

Search API Docs

You can contributing to expand the database to expand the knowledge of this library.

You can create a fork at https://github.com/RWS-Studio/open-src-library/fork.

You have to respect the syntax.

        {
          "name": "",                     -> str
          "website": "",                  -> url (http:// required)
          "repo": "",                     -> url (http:// required)
          "installation": "",             -> url (http:// required)
          "presentation": "",             -> str
          "license": "",                  -> str
            "license_url": "",            -> url (http:// required)
            "version": "",                -> str (more information : http://semver.org/)
            "tags": [ 
                "",                       -> str
                ""                        -> str
            ]
        }
